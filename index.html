Historia da Web

  • Mark I, Colossus, Eniac
  • Arpanet (1969)
  • TCP/IP
  • World Wide Web (1989)
  • Navegadores: Mosaic, Netscape, Internet Explorer, Google Chrome

O que são clients

  • Arquivos estáticos
  • Responsividade
  • Navegadores (Boas Práticas e testes)
  • W3C, W3School
  • Aplicação Web
  • Dispositivos Moveis

O que são servers

  • Arquivos, Segurança, Streaming, Emails, Web
  • Software x Hardware
  • Proxy, FireWall
  • Intranet (Rede Corporativa)
  • DNS (Domain Name System)
  • FTP (File Transfer Protocol)

Linguagem de programação

  • Server Side: Java, PHP, Ruby, C#, JavaScript...
  • Client Side: JavaScript
  • JavaScript (V8 Engine)

Introdução: HTML na prática

  • Head (informações transmitidas para o Navegador, invisível para o usuário)
  • Boddy (informações visíveis transmitidas para o Cliente)
  • Tags e Atributos: id, style, class, name...
  • Textos, Listas
  • Links (_blanck, _Safe)
